# 🚀 Fullstack-lll

Proyecto desarrollado con **Java, Spring Boot y MySQL**.

## 📌 Descripción

Aplicación full stack orientada a la gestión de datos conectada a base de datos MySQL.

## 🛠️ Tecnologías

- Java
- Spring Boot
- MySQL
- JPA / Hibernate
- Maven

## ⚙️ Configuración

Editar `application.properties`

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/sanosysalvos
spring.datasource.username=root
spring.datasource.password=tu_password
▶️ Ejecutar proyecto
mvn spring-boot:run

o ejecutar desde tu IDE.

🌐 Acceso
http://localhost:8080
